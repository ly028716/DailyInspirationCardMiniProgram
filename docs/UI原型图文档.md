# AI每日灵感卡片 - UI原型图文档

## 📱 小程序UI设计总览

### 设计原则
- **简洁优雅**：极简设计，突出内容
- **沉浸体验**：全屏卡片，减少干扰
- **个性化**：支持主题切换和字体选择
- **情感化**：温暖的色彩搭配，舒适的视觉体验

## 📱 响应式设计

### 适配方案
- **rpx单位**：微信小程序专用响应式单位
- **媒体查询**：@media screen and (min-width: 750rpx)
- **弹性布局**：flex布局为主
- **百分比布局**：辅助适配

### 断点设置
- **小屏手机**: <375rpx
- **标准手机**: 375-414rpx
- **大屏手机**: >414rpx
- **平板**: >768rpx
